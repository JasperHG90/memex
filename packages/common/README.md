# Memex Common (`memex-common`)

Shared utilities, schemas, and configuration models used across the Memex monorepo.

## Components

- **Config**: Pydantic models for configuration (`MemexConfig`).
- **Schemas**: Shared Data Transfer Objects (DTOs) for API communication.
- **Types**: Common type definitions.
- **Exceptions**: Standardized error handling.

This package ensures consistency between the Core, CLI, MCP server, and Dashboard.
