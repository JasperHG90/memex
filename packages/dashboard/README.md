# Memex Dashboard (`memex-dashboard`)

A web-based visual interface for exploring your Memex knowledge graph. Built with Reflex.

## Features

- **Knowledge Graph**: Visualize connections between entities.
- **Lineage View**: Trace how facts were derived from source documents.
- **Semantic Search**: Interactive search with filtering.
- **Vault Management**: Manage project isolation.

## Usage

Start the dashboard (requires `memex server start` running):

```bash
memex dashboard start
```
