Memex

Long-term memory for LLMs — persistent, evolving knowledge.

Why I created Memex

  • LLMs forget everything between sessions — every conversation starts from zero
  • I use many places to talk to models (architecture, code, research) — I need shared memory across Claude, Gemini, OpenCode, Craft Agents, and any future model
  • I want model functionality to be swappable — some or all components should work with open-source / local models, not locked to one provider
  • Need a system that stores memories and synthesized knowledge, but can always trace back to the source and its assets (diagrams, architecture docs) via lineage
  • Data-first philosophy: Markdown files + Postgres, not proprietary formats — your notes, always readable, never locked in

What I needed

  • Memory that does not update beliefs — it creates and updates mental models per entity, synthesizing information. Isolated per vault
  • Multiple retrieval channels: entities, memory search (synthesized facts), and direct note search
  • Structured retrieval, not just "nearest 10 vectors" — temporal, entity-aware, graph-connected
  • Full provenance: trace any mental model back to the original source document
  • Multi-vault isolation — keep work and personal knowledge separate
  • Token-efficient: only retrieve what's actually needed (JIT context)

How I use Memex

  • Claude Code + MCP: Memex runs as a tool server — Claude reads memories but does not automatically write notes
  • Firefox extension: one-click save of articles, PDFs & web pages — encrypted API key storage, automatic Markdown conversion
  • Feed it architecture docs, meeting notes, research — it extracts facts, resolves entities, builds a knowledge graph
  • Memex accurately date-stamps events relative to the publishing date — temporal queries just work
  • Background reflection synthesizes mental models overnight — next morning, Claude knows more than yesterday
  • Contradiction detection keeps knowledge fresh — new facts automatically supersede old ones

Features

What Memex does under the hood.

📥
Ingest Anything
Markdown, PDF, Word, PowerPoint, Excel, Outlook emails, web pages, or entire directories. Conversion via MarkItDown & PyMuPDF. Pluggable note templates, asset management, and batch CLI operations.
🧠
Five-Strategy Retrieval (TEMPR)
Semantic, Keyword, Graph, Temporal, and Mental Model — five strategies run in parallel, fused via Reciprocal Rank Fusion. MMR diversity filtering prunes near-duplicates.
🌳
Hierarchical Page Index
Documents are split into a structured TOC with section summaries, token estimates, and node IDs. Read a 50-page PDF section by section instead of dumping everything into context.
🔄
Incremental Extraction
Update a note and Memex diffs the content, only re-extracting changed blocks. Unchanged facts, entities, and embeddings are preserved — fast for living documents.
⚔️
Contradiction Detection
New facts are triaged for corrections. When a newer note contradicts an older one, confidence is adjusted and supersession links are recorded. Retrieval favors current information.
🪞
Reflection & Mental Models
Background reflection synthesizes observations and builds versioned mental models. Memex evolves from raw facts into structured understanding over time.
🏦
Vaults
Isolate knowledge by project, team, or topic. Policy-based ACL (reader/writer/admin) with vault-scoped API keys. Cross-vault read access for shared knowledge without write permissions.
☁️
Cloud-Native Storage
File store uses fsspec — swap between local disk, S3, and GCS with a config change. PostgreSQL + pgvector for metadata and vector search.
🤖
AI Agent Integration
First-class MCP support for Claude Code, Claude Desktop, and Cursor. 26 MCP tools, streamable HTTP transport, slim Docker image decoupled from core.
🌐
REST API & Webhooks
FastAPI server with NDJSON streaming, OpenAPI docs, policy-based auth (reader/writer/admin), vault-scoped API keys, rate limiting, and CORS extension support.
🧬
Lineage & Provenance
Trace any mental model back through observations to the original source document. Full bidirectional provenance traversal with depth control.
🦊
Firefox Extension
One-click save of articles, PDFs & web pages. Readability extraction, Markdown conversion, inline image capture. AES-GCM encrypted API key storage.
